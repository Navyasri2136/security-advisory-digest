# Sample Security Advisories

## Advisory 1
Title: CVE-2025-1234 Remote Code Execution
Severity: Critical

Description:
A remote code execution vulnerability has been identified in ExampleWeb Server v3.2.

Recommendation:
Update immediately to version 3.3 or later.

---

## Advisory 2
Title: CVE-2025-2345 SQL Injection
Severity: High

Description:
An SQL injection vulnerability allows attackers to access sensitive database information.

Recommendation:
Apply the latest security patch and validate all user inputs.

---

## Advisory 3
Title: CVE-2025-3456 Cross-Site Scripting (XSS)
Severity: Medium

Description:
Improper input sanitization may allow attackers to inject malicious scripts.

Recommendation:
Implement output encoding and update affected components.
